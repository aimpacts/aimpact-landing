# AImpact Landing Page

Landing page da AImpact — Inteligência Artificial para Empresas. Inspirada no design sofisticado do [Lumen Lab](https://www.lumenlab.ai/).

## Tecnologias

- **HTML** — Estrutura
- **CSS** — Estilização (design escuro, gradientes)
- **JavaScript** — Interatividade (menu mobile, FAQ accordion, formulário)

## Como rodar

Abra o arquivo `index.html` diretamente no navegador, ou use um servidor local:

```bash
# Com Python
python -m http.server 8000

# Com Node.js (npx)
npx serve .
```

Acesse `http://localhost:8000` (ou a porta indicada).

## Deploy (Netlify)

Arraste a pasta do projeto no Netlify ou conecte ao repositório Git. O build é estático — não é necessário comando de build.
